\examtag{可換環}
\examtag{極大イデアル}
\examtag{べき零元}

\(a\) を複素数とし，複素数体 \(\C\) 上の可換代数
\[
  A=\C[x,y]/(xy,y(y-a))
\]
を考える。

\begin{enumerate}
  \item \(A\) の極大イデアルを全て求めよ。
  \item \(A\) の各極大イデアル \(\mathfrak m\) に対して \(\dim_{\C}\mathfrak m/\mathfrak m^2\) を計算せよ。
  \item \(A\) の \(0\) でないべき零元を全て求めよ。
\end{enumerate}
