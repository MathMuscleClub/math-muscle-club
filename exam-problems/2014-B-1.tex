\(K=\F_3(T)\) とし，\(K\) 上の多項式 \(f\) を
\[
  f(X)=X^3-X
\]
と定める。\(L\) を \(f(f(X))-T\) の最小分解体とし，\(L\) に含まれる \(f(X)-T\) の最小分解体を \(M\) とする。

\begin{enumerate}
  \item \([M:K]\) を求めよ。
  \item \([L:K]\) を求めよ。
  \item \(L\) に含まれる \(K\) の \(9\) 次拡大の個数を求めよ。
  \item \(L\) に含まれる \(K\) の \(9\) 次拡大であって，以下の性質をすべてみたすものを一つあげよ。
   1. \(K\) の Galois 拡大ではない。
   2.  \(1\) のべき根で \(K\) に含まれないものを \(1\) つ以上含む。

\end{enumerate}
